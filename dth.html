01

恒温恒湿试验箱 / 高低温交变湿热试验箱

DTH-A165、DTH-B225、DTH-C408、DTH-D800、DTH-E1000 五种标准容积，支持高低温、恒定湿热与交变湿热试验。

温湿度性能
  • 温度控制范围：-70℃ ~ +150℃
  • 湿度控制范围：10%RH ~ 98%RH
  • 温度稳定度：±0.2℃
  • 湿度稳定度：±2%RH
  • 温度解析度：0.01℃
  • 湿度解析度：0.1%RH
均匀度
  • 温度均匀度：±1.0℃（-40℃ ~ +100℃）
  • 温度均匀度：±1.5℃（-40.1℃ ~ -70℃ / +100.1℃ ~ +150℃）
  • 湿度均匀度：±3%RH（中心点）
  • 部分大容积配置：±5%RH（中心点）
控制与记录
  • DTA 7寸 800 × 600 TFT 彩色触控屏
  • 程序 / 定值、故障报警与循环功能
  • RS485、RS232 远程通讯
  • SD 记录模组，可长时间记录试验数据
  • 中英文操作界面
结构特点
  • 顶部防滴露设计
  • 水电分离及溢水位、压力平衡设计
  • 水箱快速插拔，便于保养
  • 动态测试安全保护
  • 低噪音设计，标称低于 65dB
  • 独立动态超温保护系统

主要装置与使用条件

控制、循环、冷冻、加热与加湿系统的标准配置。

循环系统机械式对流系统
冷冻系统空气冷却密闭型 / 半密闭型压缩机，HFC 环保冷媒；一元式或二元式配置
加热系统平衡温度控制 P.I.D + P.L.C + A.D + S.S.R
加湿系统平衡湿度控制 P.I.D + P.L.C + A.D + S.S.R
加湿给水自动水位控制 + 可回收供给系统
控制器DTA 7寸触控式控制器
电源AC 220V / 单相 3线；AC 380V / 三相 5线
用水蒸馏水
环境温度+5℃ ~ +30℃
选配附件时序插座、CM 通讯介面、大视窗及其他项目配置
型号编码
  • 示例：DTH-A-4-15
  • A：50×55×60cm；B：50×60×75cm
  • C：60×80×85cm；D：100×80×100cm
  • E：100×100×100cm；Z：客户尺寸
  • 低温代码 0 ~ 7 对应 0℃ ~ -70℃
  • 高温代码 10 / 15 对应 +100℃ / +150℃
执行标准
  • GB/T 5170.2-2008 温度试验设备
  • GB/T 5170.5-2008 湿热试验设备
  • GB/T 2423.1-2008、GB/T 2423.2-2008
  • GB/T 2423.3-2006、GB/T 2423.4-2008
  • GJB 150.3A-2009、150.4A-2009、150.9A-2009
  • 对应 IEC 68-2-1 / 2 / 3 / 30 与 MIL-STD-810F
