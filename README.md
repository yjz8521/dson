GitHub Pages 公開網址：

https://yjz8521.github.io/dson/

## 公開站點已補齊

- `robots.txt`：允許搜尋引擎索引，並指向 sitemap
- `sitemap.xml`：提交 Google Search Console / Bing Webmaster Tools 使用
- `favicon.svg`：瀏覽器圖示
- `site.webmanifest`：站點 manifest
- `.nojekyll`：避免 GitHub Pages 使用 Jekyll 處理靜態檔案
- 首頁 SEO meta：canonical、description、keywords、Open Graph、Twitter Card
- JSON-LD 結構化資料：Organization

## 上線後建議

1. 確認 GitHub 倉庫為 Public。
2. 在 GitHub Pages 設定中選擇 `master` 分支作為發布來源。
3. 到 Google Search Console 新增網址前綴：`https://yjz8521.github.io/dson/`。
4. 提交 sitemap：`https://yjz8521.github.io/dson/sitemap.xml`。
5. 補上正式信箱與完整地址後，再更新首頁聯絡區。
